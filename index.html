üöÄ Setup Timeline

  • Step 1: Update your system sudo apt update && sudo apt upgrade -y
  • Step 2: Install Apache sudo apt install apache2 -y
  • Step 3: Allow HTTP through UFW sudo ufw allow in "Apache"
  • Step 4: Enable Apache on boot sudo systemctl enable apache2
  • Step 5: Start Apache sudo systemctl start apache2
  • Step 6: Replace Apache Default Page sudo bash -c 'cat <<EOF > /var/www/html/index.html' Paste your HTML, then: EOF
  • Step 7: Restart Apache sudo systemctl restart apache2
  • Step 8: Enable SSL with Certbot sudo apt install certbot python3-certbot-apache -y sudo certbot --apache
